# Usage

This skill supports three primary workflows.

## 1. Create a New Presentation

Use this when you have:
- a topic only
- rough notes
- structured content that should become a slide deck

Example prompts:

```text
Create a 10-slide product demo presentation with bold motion and a dark theme.
```

```text
Turn these rough notes into a clear teaching deck with calm pacing and clean visuals.
```

## 2. Convert a PPTX into HTML Slides

Use this when you want to preserve presentation content but improve the delivery format.

Example prompts:

```text
Convert this PPTX into HTML slides with cinematic transitions and strong typography.
```

```text
Take this PowerPoint and turn it into a clean HTML presentation for a technical keynote.
```

## 3. Enhance an Existing Deck

Use this when you already have HTML slides and want stronger presentation quality.

Example prompts:

```text
Improve this existing HTML presentation and make it feel more keynote-like.
```

```text
Keep the content but reduce text density, improve pacing, and add better transitions.
```

## When to Enable Pro Features

### Fragment reveals
Use when you want:
- speaker-controlled pacing
- step-by-step explanation
- audience focus on one point at a time

### Settings panel
Use when you want:
- live control during presenting
- appearance switching
- page management and quick tuning

### Drawing mode
Use when you want:
- annotation during demos or teaching
- emphasis while presenting live

### Data animations
Use when you want:
- large metrics
- counters
- chart-driven storytelling

## What Good Output Looks Like

A strong result should usually have:
- clear slide pacing
- readable visual hierarchy
- no scrolling inside slides
- large enough visuals for presentation use
- a stronger sense of presentation than a generic webpage
