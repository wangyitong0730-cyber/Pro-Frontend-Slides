## 1. Product Launch Deck

**Best for**
- new product announcements
- feature launches
- founder-led demos

**What the deck should feel like**
- bold
- polished
- high-confidence
- visually memorable

**Typical ingredients**
- strong title slide
- section breaks with clear pacing
- large product metrics
- feature comparisons
- demo flow slides
- closing CTA slide

**Recommended presets**
- Bold Signal
- Electric Studio
- Creative Voltage

**Best pro features**
- fragments
- transitions
- theme switching
- drawing mode for live demos

## 6. Existing Deck Upgrade

**Best for**
- old internal decks
- plain HTML slides
- exported PPT content that feels flat

**What the upgrade should improve**
- visual hierarchy
- pacing
- consistency
- spacing
- readability
- presentation controls

**Typical changes**
- split overloaded slides
- enlarge metrics and diagrams
- improve cover and section slides
- add a more coherent theme system
- introduce fragments or transitions where useful

**Recommended presets**
- reuse the closest existing mood, then refine

**Best pro features**
- depends on what the original deck lacks most

## Choosing the Right Output

If the user wants:
- **maximum energy** → lean into bolder presets and stronger transitions
- **clarity first** → use restrained motion and cleaner editorial structure
- **live speaking control** → prioritize fragments, controls, and drawing mode
- **metric impact** → prioritize large numbers, chart treatment, and spacing discipline
- **a lighter deliverable** → keep the feature set minimal and avoid loading every advanced system
