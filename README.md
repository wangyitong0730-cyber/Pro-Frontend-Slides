# Frontend Design Skill — Pro Edition

> **The original skill taught AI to stop making ugly sites. This fork teaches it to make slides that make audiences forget they're looking at HTML.**

## What's New

The upstream `frontend-design` skill generates gorgeous pages. But when you need **slide decks that rival Keynote**, you need more firepower.

| Feature | What It Does |
|---|---|
| **Fragment Reveal** | `Space` to unveil content — 4 entrance variants (fade-up/left/right/scale/blur) |
| **Page Transitions** | 4 cinematic effects (Fade/Slide/Zoom/Flip) replacing scroll-snap |
| **4 Live Themes** | Dark/Light/Warm/Ocean — press `T` to cycle |
| **Settings Panel** | Scene presets + feature toggles + appearance + page management |
| **Page Manager** | Drag-to-reorder, edit mode, delete pages |
| **Drawing Canvas** | Press `D` — 5 colors, 3 brush sizes, full-screen annotation |
| **Data Animations** | Number countups (ease-out quart) + bar chart growth |
| **PDF Background Fix** | `print-color-adjust: exact` — hard-won wisdom |
| **Selective Features** | Cherry-pick pro features, no longer all-or-nothing |
| **10 Visual Impact Rules** | Battle-tested design laws (no text-only slides, data big, charts bigger...) |
| **Advanced Interactions** | Spotlight cursor, 3D tilt cards, SVG stroke drawing, staggered entrances |

## Keyboard Shortcuts

```
Space  → reveal next fragment
T      → cycle themes
D      → draw on slides
←/→    → navigate with transitions
```

## Philosophy

Most AI-generated slides look like a committee of placeholder text designed them. This skill enforces **10 Visual Impact Rules** from real presentation disasters — because "just export to PDF" is not a design philosophy.

## Usage

Works as a [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skill. Drop into your skills directory and invoke via `/frontend-slides`.
